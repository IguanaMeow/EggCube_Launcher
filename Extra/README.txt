These files are used to start AssaultCube and connect to tm's masterserver.

For Windows Vista, 7, 8, 8.1, and 10 users use EggCube_Win7.bat

For Windows 2000 and XP users use EggCube_WinXP.bat

For Linux and BSD users use EggCube_Linux.sh

For OSX users add: -mgc.eggcube.party --masterport=28760 --mastertype=0
To your launcher's command line.
